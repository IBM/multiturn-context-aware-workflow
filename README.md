- [`code/`](code/)

  - [`code/.env.example`](code/.env.example) – example environment variables
  - [`code/Dockerfile`](code/Dockerfile) – container build definition

  - [`code/nginx-combined.conf`](code/nginx-combined.conf) – nginx configuration
  - [`code/supervisord.conf`](code/supervisord.conf) – process management config
  - [`code/.gitignore`](code/.gitignore) – ignore rules
- [`README.md`](README.md) – project overview
